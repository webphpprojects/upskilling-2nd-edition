# Task Manager
Self learning project


## Installation
Clone repository

## How to launch

```bash
docker-compose up
docker exec -it tm_php composer install
---
docker-compose rm --stop --force
```

## Run Behat
```bash
cd taskmanager
vendor/bin/behat
```
